# Flappy Bird em Python

## Visão Geral
Projeto desenvolvido em **Python** que recria o jogo **Flappy Bird**, com foco no estudo de lógica de programação, controle de eventos, física básica e manipulação de gráficos. O projeto demonstra domínio de estruturas fundamentais da linguagem aplicadas a um jogo interativo.

## Objetivo do Projeto
Este projeto foi criado com o objetivo de praticar:
- Lógica de programação
- Estruturas de controle e loops
- Manipulação de eventos do teclado
- Atualização de estados em tempo real
- Organização de código em aplicações interativas

## Tecnologias Utilizadas
- Python  
- Pygame  

## Como Funciona
- O jogador controla o personagem através de comandos do teclado.  
- O jogo executa um loop contínuo responsável por:
  - Atualizar a posição do personagem  
  - Detectar colisões  
  - Controlar pontuação e estados do jogo  
- A lógica de física simples simula gravidade e impulso do salto.
